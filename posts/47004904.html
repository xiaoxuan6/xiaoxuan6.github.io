Docker logging driver

对 Docker 而言，其默认的 logging driver 是 json-file，如果在启动时没有特别指定，都会使用这个默认的 logging driver。

json-file 会将我们在控制台通过 docker logs 命名看到的日志都保存在一个 json 文件中，我们可以在服务器 Host 上的容器目录中找到这个 json 文件。
